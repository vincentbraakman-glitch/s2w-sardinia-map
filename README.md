# S2W Sardinia & Corsica Sailing Map

Interactive Leaflet map for the 7-day Sardinia & Corsica sailing route (Route B — Sardinia anchor finish).

Built for Doni Belau / InRetreat Sardinia Sep 2027.

- Vanilla JS + Leaflet 1.9.4
- Single `index.html`, no build
- Deployed via Vercel
- Iframe-embeddable (CSP allows all frame-ancestors)

## Route
Porto Rotondo → Golfo Aranci → Liscia di Vacca → Capo d'Orso → Cannigione → Bonifacio (Corsica) → Lavezzi → La Maddalena → Spargi → Caprera Bay → Porto Rotondo
